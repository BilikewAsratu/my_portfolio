# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/my-portfolio.git
```

Go into the project

```bash
cd my-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
